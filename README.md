# NyxHyperion
Processing Night-Time Light Images from VIIRS

# Requirements
* Anaconda with Python36
```bash
conda env create -n ntl -f ntl.yml
```
__Windows__ activate the environment with
```bash
activate ntl
```
__Linux__ activate the environment with
```bash
source activate ntl
```
